🔗 How Players Connect

This server runs entirely in your browser. Players connect via WebRTC — no IP address needed, works on restricted networks and Chromebooks.

  1. Start the server (top-right button), then click Generate Offer below.
  2. Copy the offer code and send it to the player (Discord, email, etc.).
  3. Player opens their Eaglercraft/EaglercraftX client and pastes the offer.
  4. Player sends back an answer code — paste it in the Answer box.
  5. Connection established! Player appears in the Players tab.
