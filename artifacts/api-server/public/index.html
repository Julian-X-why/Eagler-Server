Static Hosting Guide

You're on a static host. WebRTC works fine — the IP/WS relay requires Node.js.

  1. Download the ZIP (⬇ ZIP button in the toolbar) and extract it.
  2. Run node server.js on a server or locally for full relay support.
  3. Players without IP access can always connect via WebRTC Offer/Answer above.
