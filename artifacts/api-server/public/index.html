Static Hosting Guide GitHub Pages, Netlify, etc.

You're running EaglerNet from a static host. WebRTC works fine — the IP/WS relay requires the Node.js server.

  1. Copy the public/ folder to any static host (GitHub Pages, Netlify, Cloudflare Pages, etc.).
  2. For IP connections: clone the repo and run pnpm install && pnpm run dev on a server.
  3. Players without IP access can always connect via WebRTC Offer/Answer above.

📋 Quick Plugin Reference

BOTTLE.register({
  id: 'my-plugin', name: 'My Plugin',
  version: '1.0.0', author: 'You',
}, {
  'player.join'({ player }) {
    player.sendMessage('§aWelcome, ' + player.username + '!');
  },
  'player.chat'({ player, message }) { /* return false to cancel */ },
  'server.tick'({ tick })            { /* runs every 50ms */ },
  command(player, cmd, args) {
    if (cmd !== 'hello') return false;
    player.sendMessage('§aHello!'); return true;
  },
});
